About Emmet

Emmet is a plugin for code editors like Sublime, Dreamweaver, and VSC. Emmet is consistent across most code editors. It uses CSS syntax.

Cheat Sheet for Emmet

Hint: check the console log.

!
This sets up an html file.

li>a
This creates a list with an link.

div.container
This creates a div with a class called container.

#container
This creates a div with an id called container.

#container.block-1.block-2.block-3.
This creates a div with an id called container with multiple blocks with the class of 1, 2, 3.

ul>li
This creates an unordered list. Of note the > means that list goes inside the ul element.

h1{Noctis is the best protagonist!}+h5{Gladio is still my favorite though.}
This creates an h1 element containing the text inside the brackets with a sibling element (h5 containing it's own text).

Expansion

div.block+h1{"Never fear, Prompto's here!"}+p{"I never miss... A photo op."}*3
This + creates an expands the block.

.blocks*2>h2{"Blondie!"}+img+p>lorem

Grouping and Climbing

.container>h1{Course}+p>lorem>a{Ignis though}^p{"That's it! I've come up with a new recipeh!"}
This is climbing signalled with the use of ^.

.container>h1{Course}+p>lorem>a{Ignis though}+p{"That's it! I've come up with a new recipeh!"}
This is grouping, signalled by ().

Numbers

.container>img[class="thumbnail $"]*5
This creates 5 images with numbered classes.

.container>img[class="thumbnail_$"]*5
This creates 5 images with numbered classes with an underscore.

.container>img[class="thumbnail $@5"]*5
This creates 5 images with numbered classes with the count starting at 5.

.container>img[class="thumbnail $@5-"]*5
This creates 5 images with numbered classes with the class counting down from 5.

Toggling Comments

To comment out using emmet, simply select the text and ctrl (or cmd) + forward slash. Using this same combination will undo the comment. This same command works for css.

Attributes

.container>img[src="images/xyz/100x100"]
Note: This will not work, img is bogus.
